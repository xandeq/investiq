Redis cache key convention (use exactly these):
- `market:quote:{TICKER}` — e.g., market:quote:PETR4
- `market:macro:{indicator}` — e.g., market:macro:cdi, market:macro:selic, market:macro:ipca, market:macro:ptax_usd
- `market:fundamentals:{TICKER}` — e.g., market:fundamentals:PETR4
- `market:historical:{TICKER}` — e.g., market:historical:PETR4

TTL values (seconds):
- Quotes: 1200 (20 min — slightly longer than 15-min refresh to avoid cold gaps)
- Macro: 25200 (7h — longer than 6h refresh task interval)
- Fundamentals: 86400 (24h)
- Historical: 86400 (24h)

Stale data policy: when Redis key is empty (None), return last known value if available,
OR return {"data_stale": true, "reason": "cache_empty"} in the response.

brapi.dev API (locked per CONTEXT.md):
- Base URL: https://brapi.dev/api
- Quote: GET /quote/{ticker}?token={BRAPI_TOKEN}
  Response: {"results": [{"symbol": "PETR4", "regularMarketPrice": 38.50, ...}]}
- Quote list: GET /quote/list?token={BRAPI_TOKEN}&search={tickers_csv}
- Fundamentals: GET /quote/{ticker}?modules=defaultKeyStatistics,financialData&token={BRAPI_TOKEN}
  Response: {"results": [{"defaultKeyStatistics": {"priceToBook": 1.2, ...}, "financialData": {...}}]}
- Historical: GET /quote/{ticker}?range=1y&interval=1d&token={BRAPI_TOKEN}
  Response: {"results": [{"historicalDataPrice": [{"date": ..., "open": ..., "high": ..., "low": ..., "close": ..., "volume": ...}]}]}
- IBOVESPA: GET /quote/%5EBVSP?token={BRAPI_TOKEN} (ticker is ^BVSP, URL-encoded)
- NOTE: brapi.dev Startup plan (R$59.99/mo) required for production.
  Free tier: 15,000 req/month, 30-min delay. Startup: no rate limit published, 15-min delay.
  Implement 200ms sleep between batch requests to avoid throttling.

python-bcb SGS codes (locked per RESEARCH.md):
- CDI: series 12 (daily rate, annualized)
- SELIC: series 11 (daily target rate)
- IPCA: series 433 (monthly CPI)
- PTAX USD: use bcb.get_currency("USD") or SGS series 1 (BRL/USD)

AWS Secrets Manager:
- brapi.dev token: secret ID `tools/brapi` (create if doesn't exist), key `BRAPI_TOKEN`
- Fetch at worker startup via boto3, cache in process memory (not re-fetched per request)

    **`backend/app/modules/market_data/tasks.py`:**

    Tasks are synchronous (Celery default). Use `celery_app` from `app.celery_app`. Import `BrapiClient` and `fetch_macro_indicators`. Use `redis.Redis.from_url(os.environ["REDIS_URL"])` (sync Redis client) inside the task to write cache.

    ```python
    from app.celery_app import celery_app
    import redis as redis_lib
    import json
    import os

    # Tickers to refresh — in production, read from DB (all tickers in active portfolios)
    # For Phase 2, use a hardcoded seed list that Celery refreshes
    DEFAULT_TICKERS = ["PETR4", "VALE3", "ITUB4", "BBDC4", "WEGE3", "MGLU3", "ABEV3", "BOVA11"]

    @celery_app.task(bind=True, max_retries=3, default_retry_delay=60)
    def refresh_quotes(self):
        """Fetch B3 quotes from brapi.dev and write to Redis cache."""
        r = redis_lib.Redis.from_url(os.environ.get("REDIS_URL", "redis://localhost:6379/0"))
        client = BrapiClient()
        try:
            quotes = client.fetch_quotes(DEFAULT_TICKERS)
            for q in quotes:
                key = f"market:quote:{q['symbol']}"
                r.set(key, json.dumps(q), ex=1200)
            # Also fetch IBOVESPA
            ibov = client.fetch_ibovespa()
            r.set("market:quote:IBOV", json.dumps(ibov), ex=1200)
        except Exception as exc:
            raise self.retry(exc=exc)

    @celery_app.task(bind=True, max_retries=3, default_retry_delay=300)
    def refresh_macro(self):
        """Fetch macro indicators from BCB and write to Redis cache."""
        r = redis_lib.Redis.from_url(os.environ.get("REDIS_URL", "redis://localhost:6379/0"))
        try:
            macro = fetch_macro_indicators()
            for key_suffix, value in macro.items():
                if key_suffix == "fetched_at":
                    continue
                r.set(f"market:macro:{key_suffix}", str(value), ex=25200)
            r.set("market:macro:fetched_at", macro["fetched_at"], ex=25200)
        except Exception as exc:
            raise self.retry(exc=exc)
    ```

    **`backend/app/modules/market_data/router.py`:**

    FastAPI router with prefix `/market-data`. Endpoints use `get_authed_db` dependency for RLS, and inject `MarketDataService` with an async Redis client. Use `from app.core.config import settings` for REDIS_URL.

    ```python
    from fastapi import APIRouter, Depends
    from app.core.middleware import get_authed_db  # ensures auth
    from app.modules.market_data.service import MarketDataService
    from app.modules.market_data.schemas import MacroCache, FundamentalsCache, HistoricalCache

    router = APIRouter()

    def _get_market_service() -> MarketDataService:
        import redis.asyncio as aioredis
        from app.core.config import settings
        r = aioredis.from_url(settings.REDIS_URL)
        return MarketDataService(r)

    @router.get("/macro", response_model=MacroCache)
    async def get_macro(service: MarketDataService = Depends(_get_market_service), _=Depends(get_authed_db)):
        return await service.get_macro()

    @router.get("/fundamentals/{ticker}", response_model=FundamentalsCache)
    async def get_fundamentals(ticker: str, service=Depends(_get_market_service), _=Depends(get_authed_db)):
        return await service.get_fundamentals(ticker)

    @router.get("/historical/{ticker}", response_model=HistoricalCache)
    async def get_historical(ticker: str, service=Depends(_get_market_service), _=Depends(get_authed_db)):
        return await service.get_historical(ticker)
    ```
