       a) EPS history (D-12): For each year in income_history (up to 5):
          - `eps = net_income / shares_outstanding`
          - `revenue` from same year
          - `yoy_growth = (eps_current - eps_previous) / abs(eps_previous)` (None for oldest year)
          - Calculate `eps_cagr_5y` if >= 2 years: `(newest/oldest)^(1/n) - 1` (None if oldest EPS <= 0)

       b) Revenue history (D-14): extract `totalRevenue` per year alongside EPS

       c) Accrual ratio (D-13): `(net_income - operating_cash_flow) / (total_debt + total_cash)`
          - Use most recent year income_history[0] netIncome and cashflow_history[0] operatingCashFlow
          - Denominator: `total_debt + total_cash` from fundamentals (proxy for total assets)
          - Flag as approximate in data_completeness if using proxy
          - Thresholds: < 0.20 good, 0.20-0.40 moderate, > 0.40 poor

       d) FCF conversion (D-13): `free_cash_flow / net_income`
          - Use most recent year values
          - None if net_income <= 0
          - Thresholds: > 0.80 good, 0.50-0.80 moderate, < 0.50 poor

       e) Earnings quality flag (D-13): `assess_earnings_quality(accrual_ratio, fcf_conversion) -> "high" | "medium" | "low"`
          - high: accrual < 0.20 AND fcf_conversion > 0.80
          - low: accrual > 0.40 OR fcf_conversion < 0.50
          - medium: everything else (including insufficient data)
