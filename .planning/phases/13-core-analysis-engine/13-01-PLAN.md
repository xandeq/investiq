    Create `backend/app/modules/analysis/dcf.py` — DCF calculation per D-01, D-02, D-03, D-04, D-05, D-10, D-11:

    1. `ERP_BRAZIL = 0.07` constant

    2. `calculate_wacc(selic, beta, debt, equity, tax_rate=0.34) -> float`:
       - `Ke = selic + beta * ERP_BRAZIL`
       - `Kd = selic + 0.02` (simplified debt cost spread)
       - `WACC = Ke * (E/(D+E)) + Kd * (1-T) * (D/(D+E))`
       - If beta is None, use 1.0 (market average)
       - If total_capital <= 0, return Ke

    3. `calculate_dcf(fcf_current, shares_outstanding, growth_rate, wacc, terminal_growth) -> dict`:
       - 5-year explicit FCF projections: `FCF_t = FCF_{t-1} * (1 + growth_rate)`
       - Present value each year: `PV_t = FCF_t / (1 + WACC)^t`
       - Terminal value: `TV = FCF_5 * (1 + g_terminal) / (WACC - g_terminal)`
       - PV of terminal: `PV_TV = TV / (1 + WACC)^5`
       - Enterprise value = sum(PV_FCFs) + PV_TV
       - Fair value per share = EV / shares_outstanding
       - Raise ValueError if WACC <= terminal_growth
       - Return: `fair_value`, `enterprise_value`, `pv_stage1`, `pv_terminal`, `terminal_value`, `projected_fcfs` (list with year/fcf/present_value), `assumptions`

    4. `calculate_dcf_with_sensitivity(fcf_current, shares_outstanding, growth_rate, wacc, terminal_growth, net_debt) -> dict`:
       - 3 scenarios per D-04:
         - low (bear): growth_rate - 0.02, wacc + 0.02
         - base: as-is
         - high (bull): growth_rate + 0.02, wacc - 0.02 (floor at terminal_growth + 0.01)
       - For each: equity_value = EV - net_debt, fair_value = equity_value / shares
       - Return: `fair_value`, `fair_value_range` (low/high), `upside_pct`, `scenarios`, `key_drivers`, `projected_fcfs` (from base)
       - `key_drivers` from `_identify_key_drivers()`: list of strings describing spread and sensitivity

    5. `estimate_growth_rate(cashflow_history, default=0.05) -> float`:
       - CAGR of last 5 years FCF
       - Fallback to default if <3 years, negative FCF, or CAGR outside 0-30%
