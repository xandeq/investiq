```python
async def check_analysis_rate_limit(tenant_id: str, plan_tier: str) -> tuple[bool, int]:
    """Check per-request rate limit. Returns (allowed, retry_after_seconds)."""
```
- Use `sync_redis.from_url(os.environ.get("REDIS_URL", "redis://redis:6379/0"))` — lazy singleton pattern
- Key: `f"analysis:rate_limit:{tenant_id}"`
- Window: free=300s, pro=60s, enterprise=60s
- Limits: free=1, pro=1, enterprise=100
- On rejection, return `(False, ttl_remaining)` where ttl_remaining is the Redis key TTL
- On allow, `INCR` key and `EXPIRE` if new key
