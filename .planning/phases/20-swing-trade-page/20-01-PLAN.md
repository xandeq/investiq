```python
class SwingSignalsResponse(BaseModel):
```

<action>
Create `backend/app/modules/swing_trade/service.py`:

Key functions:

1. `async def compute_signals(redis_client, portfolio_tickers: list[str]) -> SwingSignalsResponse`:
   - Import RADAR_ACOES from opportunity_detector.radar
   - For each stock in RADAR_ACOES: read `market:quote:{TICKER}`, `market:historical:{TICKER}`, `market:fundamentals:{TICKER}` from Redis
   - Compute 30d high: `max(p.high for p in historical.points if p.date >= thirty_days_ago_epoch)`
   - Compute discount: `(current_price - high_30d) / high_30d * 100`
   - BUY signal: discount < -12 AND (dy is None or dy > 5)
   - For stocks in portfolio_tickers: mark in_portfolio=True, include quantity
   - Split results: portfolio_signals (in_portfolio=True), radar_signals (in_portfolio=False, sorted by discount ascending)
   - Use MarketDataService for async Redis reads

2. `async def get_operations(db, tenant_id, status_filter) -> OperationListResponse`:
   - Query swing_trade_operations WHERE tenant_id = tenant_id AND deleted_at IS NULL
   - Filter by status if provided
   - Enrich each open operation with current_price from Redis:
     - pnl_brl = (current_price - entry_price) * quantity
     - pnl_pct = (current_price - entry_price) / entry_price * 100
     - days_open = (now - entry_date).days
     - target_progress_pct = (current_price - entry_price) / (target_price - entry_price) * 100 if target_price set
   - Add SELL signal to OperationResponse if pnl_pct >= 10
   - Add STOP signal if current_price <= stop_price

3. `async def create_operation(db, tenant_id, data: OperationCreate) -> SwingTradeOperation`

4. `async def close_operation(db, tenant_id, op_id, data: OperationClose) -> SwingTradeOperation`:
   - Set status="closed", exit_price, exit_date (default now)

5. `async def delete_operation(db, tenant_id, op_id)`:
   - Soft delete: set deleted_at = now

```python
router = APIRouter()

# Dependency: async Redis client (same pattern as portfolio)
def _get_redis():
    import redis.asyncio as aioredis
    from app.core.config import settings
    return aioredis.from_url(settings.REDIS_URL, decode_responses=False)

# GET /swing-trade/signals
@router.get("/signals")
async def get_signals(
    request: Request,
    db: AsyncSession = Depends(get_authed_db),
    tenant_id: str = Depends(get_current_tenant_id),
    redis=Depends(_get_redis),
    current_user: dict = Depends(get_current_user),
):
    # Get user portfolio tickers
    portfolio_svc = PortfolioService()
    positions = await portfolio_svc.get_positions(db, tenant_id, redis)
    portfolio_tickers = [p.ticker for p in positions]
    # Compute signals
    return await compute_signals(redis, portfolio_tickers)

# GET /swing-trade/operations
# POST /swing-trade/operations
# PATCH /swing-trade/operations/{id}/close
# DELETE /swing-trade/operations/{id}
```
