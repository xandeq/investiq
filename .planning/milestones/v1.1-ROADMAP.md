# Milestone v1.1: Onde Investir

**Status:** ✅ SHIPPED 2026-03-28

## Overview

Transformar o InvestIQ em um advisor de alocação — o usuário informa quanto tem para investir e a plataforma orienta como alocar entre ações, FIIs e renda fixa. Entregou: infraestrutura de dados globais (TaxEngine, tabelas screener, pipelines Celery), screener Goldman Sachs funcional, e wizard "Onde Investir" com 3 etapas e validação CVM.

## Milestone Summary

**Key Accomplishments:**
1. Global data infrastructure: 4 PostgreSQL tables + TaxEngine IR regressivo + GRANT-based access (no per-tenant RLS for global data)
2. 3 autonomous Celery beat pipelines refresh screener universe (~900 tickers), FII metadata, and renda fixa catalog daily
3. Screener Goldman Sachs shipped and working in production — 10 ações returned per run
4. Wizard "Onde Investir" backend: async Celery job, AI provider call, CVM disclaimer, portfolio delta output
5. 3-step frontend wizard: Valor → Prazo → Perfil with progress indicator, back navigation, CVM amber card first
6. Billing transactional emails (welcome, payment-received, payment-failed, cancellation) via Brevo
7. Correpy brokerage fee distributed proportionally across transactions (bug fix)
8. Landing page: replaced /login redirect with full marketing page (features, pricing, FAQ)

**Key Decisions:**
- `get_global_db` FastAPI dependency for screener/catalog tables — NOT tenant-scoped `get_db`
- Redis namespaces: `screener:universe:{TICKER}`, `tesouro:rates:{BOND_CODE}`, `fii:metadata:{TICKER}`
- TaxEngine IR rates in DB (not constants) — LCI/LCA exemption reform pending 2026
- Tesouro Direto via ANBIMA API (old JSON endpoint 404 since Aug 2025)
- CDB/LCI/LCA: reference ranges in `fixed_income_catalog` table, UI says "taxas de referência de mercado"
- SQLite returns naive datetimes for `DateTime(timezone=True)` — normalized with `replace(tzinfo=utc)` in plan_gate.py

**Issues Resolved:**
- VPS crash loop (ClearDesk OOM → 342% CPU → Hostinger parou o VPS) — ClearDesk scaled to 0
- Worker crash loop on startup — swap 2GB criado, uvicorn single-process
- BRAPI_TOKEN not passed to containers — added to docker-compose.yml
- Screener dispatch error — usar `celery_app.connection_for_write()` explicitamente
- Frontend OOM during build — `NODE_OPTIONS=--max-old-space-size=768` in Dockerfile
- postgres + redis containers stopped after VPS restart — fixed by `docker compose up -d`

**Issues Deferred (to v1.2+):**
- Phases 8 (Screener completo), 9 (Comparador RF vs RV), 10 (Simulador de Alocação) — planejadas mas não executadas
- FII screener (SCRF requirements) — not built
- Renda fixa catalog frontend (RF requirements) — catalog exists in DB, UI missing
- Comparador retorno líquido (RF-04, RF-05) — not built
- Simulador de alocação (SIM requirements) — not built

**Technical Debt:**
- deploy via docker cp (apt-get falha na rede do VPS durante builds) — workaround permanente até migrar infra
- Screener não tem fase formal nem SUMMARY — contexto só nos commits
- Phase 8, 9, 10 planejadas no ROADMAP mas nunca executadas — remover ou reprojetar em v1.2
